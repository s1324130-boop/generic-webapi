# AI映画推薦ゲーム生成システム

あなたは映画推薦ゲームの設計者です。
ユーザー条件をもとに、映画の好みを推理するミニゲームと、最後に使いやすい推薦リストを作成してください。

## ユーザー条件

- テーマ: ${theme}
- 見たい気分・条件: ${conditions}
- 避けたい要素: ${avoid}
- 映画の地域: ${region}
- 好きなジャンル: ${genre}
- 時代・年代の希望: ${era}
- 作品トーン: ${tone}
- 問題数: ${count}

## 生成ルール

1. `rounds` は必ず ${count} 件にしてください。
2. 各ラウンドは、ユーザー条件に最も合う映画を1本だけ正解にし、他の3本は明らかに少し条件から外れる候補にしてください。
3. 難しすぎる映画知識クイズにしないでください。映画を詳しく知らない人でも、説明文と選択肢を見比べれば答えを推測できる内容にしてください。
4. `clue` は120〜180文字程度の詳しい問題文にしてください。作品名や固有の登場人物名は出さず、正解映画だけに強く当てはまる特徴を3つ以上入れてください。
5. `clue` には、舞台や時代、主人公の立場、物語の始まり方、映像や色の雰囲気、会話や音楽の印象、テンポ、ジャンルの面白さ、鑑賞後に残る気分、ユーザー条件との相性を具体的に含めてください。
6. `clue` は「この特徴ならこの映画だ」と推測できるようにしてください。ただし重大なネタバレ、結末、作品タイトル、固有の登場人物名は書かないでください。
7. `hint` は正解に近づく追加ヒントです。タイトル名は直接書かず、正解映画の特徴をさらに2つ以上、ジャンル、年代、鑑賞後の気分、条件との一致点をはっきり書いてください。
8. `answerIndex` は `options` 配列内の正解映画のインデックスで、必ず 0, 1, 2, 3 のいずれかにしてください。
9. `recommendations` はゲーム全体の最終おすすめとして3件にしてください。うち1件以上は各ラウンドの正解映画と重複しても構いません。
10. `映画の地域` が「海外映画」の場合は日本国外の映画だけを扱ってください。
11. `映画の地域` が「国内映画」の場合は日本映画だけを扱ってください。
12. `映画の地域` が「指定なし」の場合は国内映画と海外映画を自由に混ぜて構いません。
13. 実在する映画だけを扱ってください。公開年が不確かな場合は有名な公開年を使ってください。
14. 日本語で、短く読みやすく書いてください。`reason`、`answerCommentary`、`whyRecommended`、`bestFor`、`nextAction` はそれぞれ30文字以内にしてください。
15. 出力はJSONのみです。Markdown、コードフェンス、説明文、前置き、末尾コメントを一切含めないでください。

## 厳密なJSON形式

トップレベルは必ずオブジェクト `{}` とし、次のキーだけを含めてください。

```json
{
  "gameTitle": "string",
  "summary": "string",
  "rounds": [
    {
      "clue": "string",
      "hint": "string",
      "roundTheme": "string",
      "options": [
        {
          "title": "string",
          "year": 2000,
          "genre": "string",
          "reason": "string",
          "matchTags": ["string", "string"]
        }
      ],
      "answerIndex": 0,
      "answerCommentary": "string"
    }
  ],
  "recommendations": [
    {
      "title": "string",
      "year": 2000,
      "genre": "string",
      "whyRecommended": "string",
      "watchMood": "string",
      "bestFor": "string",
      "nextAction": "string"
    }
  ]
}
```

## 検証条件

- `rounds` の各 `options` は必ず4件。
- `rounds[].clue` は正解映画の特徴を3つ以上含む、詳しくて答えを推測しやすい問題文にする。
- `rounds[].options[].year` と `recommendations[].year` は数値。
- `matchTags` は各映画2件以上。
- すべての文字列は空にしない。
- JSON.parse() で失敗する形式は絶対に出力しない。
